# Gemini Agent Guide

## Papel
Gemini é usado como orquestrador contextual, pesquisa rápida e execução de tarefas seguras com foco em custo baixo.

## Regras
- Preferir tool use restrito.
- Reduzir contexto desnecessário.
- Usar dashboard e handoff como memória compartilhada.
- Não assumir que roadmap antigo continua válido sem checar o código atual.

## Uso esperado
- roteamento
- pesquisa
- revisão rápida
- apoio a decisões com baixo custo
