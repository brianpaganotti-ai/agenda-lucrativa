# GitHub Project Fields

Campos sugeridos para o dashboard do projeto.

- `Agent Owner` → Claude / Gemini / GPT / Human
- `Work Type` → Arquitetura / Implementação / Review / Pesquisa / Deploy
- `Risk` → baixo / médio / alto
- `Execution Mode` → auto / review-required / manual-only
- `Provider` → gemini / claude / gpt / mixed
- `Repo Scope` → main / lab
- `Status` → backlog / ready / running / blocked / review / done
- `Next Check` → data
- `Token Sensitivity` → baixa / média / alta

## Regra
Dashboard mostra estado operacional; detalhes e instruções ficam nos arquivos em `ops/` e `agents/`.
